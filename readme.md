# 一、项目概述

## 1.1 项目名称

**Arxiv 论文检索与收藏系统**

## 1.2 目标

构建一个基于 arXiv API 的 Web 论文检索与收藏系统，支持：

* 用户在前端输入检索条件，查询 arXiv 论文
* 前端展示查询结果（标题、作者、分类、摘要、链接等）
* 用户可以选择一些论文保存到本地 MySQL 数据库，作为个人/项目库
* 后端通过封装好的 `arxiv_client.py` 调用 arXiv API，控制节奏，避免限流（429）

**不做：** 全量爬虫、增量同步全站、复杂任务队列，目前只做「在线搜索 + 局部收藏」。

---


默认技术栈：

* 前端：React（可后续选 React Router / Zustand / Redux 等）
* 后端：Python + FastAPI（示意），持久层：MySQL
* arXiv 访问：使用你已经验证可跑通的 `arxiv_client.py` 模块（上一条消息那套）
